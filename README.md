This project performs Exploratory Data Analysis (EDA) on a sample tabular dataset using Python, with the goal of understanding data structure, missing values, and basic feature distributions. The workflow is split across two files: eda.py, which contains a reusable data_clean() function for handling missing data and dropping low-quality columns, and dataset_EDA.ipynb, which demonstrates the complete analysis process in a Jupyter Notebook. The notebook imports the cleaning function, creates a sample employee-style dataset, generates a cleaned copy, inspects the dataset using head(), info(), and shape checks, evaluates missing values, and produces descriptive statistics. It also includes visualizations such as a correlation matrix, histograms, and boxplots to explore relationships, distributions, and potential outliers in the cleaned data. The sample dataset contains 15 records and 6 columns, including fields such as ID, Name, Age, Salary, City, and Country, making the project a simple and practical example of data cleaning and introductory EDA in Python.
